## 项目简介

这是我的个人主页项目，基于 MDUI2 框架构建的响应式静态网站。集成了个人展示、项目导航、一言、音乐播放等功能的个人主页。

在线访问：[maker.ct.ws](https://maker.ct.ws)

## ✨ 功能特性

- **响应式设计**：完美适配桌面端和移动端
- **主题切换**：支持明暗主题自动切换
- **一言集成**：实时显示随机美文佳句
- **项目展示**：清晰展示个人项目和技术栈
- **音乐播放**：内置背景音乐控制
- **访客统计**：实时显示网站访问量
- **运行时间**：显示网站稳定运行时长

## 🛠 技术栈

- [MDUI](https://www.mdui.org/) - 前端 Material Design 组件库
- [Hitokoto](https://hitokoto.cn/) - 一言 API 服务
- [jsdelivr](https://www.jsdelivr.net/) - CDN 加速服务
